MÉTODO COSMIC (Common Software Measurement International Consortium)
---------------------------------------------------------------------
El método COSMIC mide el tamaño funcional del software basándose en
movimientos de datos:
- E (Entry): Entrada de datos desde el usuario
- X (Exit): Salida de datos hacia el usuario
- R (Read): Lectura de datos persistentes
- W (Write): Escritura de datos persistentes

Cada movimiento de datos = 1 CFP (COSMIC Function Point)
